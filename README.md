# Dot-Files

My personal collection of dot files. 

Made using [chezmoi](https://www.chezmoi.io). Clone for the (objectvely) perfect Arch Linux experience.

Includes files for:

- Zsh
- Nvim
- Powerlevel10k

Important points:

- Use `cz add {file path}` to add new files. 
- Use `cz re-add` to automatically reflect changes made to files in their original locations in the chezmoi folder.
